### 2026-03-12：小地图实现统一

**决策：** 保留 MinimapRenderer.ts，重构 MiniMap.tsx

**原因：**
1. MinimapRenderer 支持动态更新
2. Canvas 2D API 更轻量
3. 避免 Pixi.js 重复初始化
4. 减少 Canvas 数量（3→2）

**实施：**
```typescript
// MiniMap.tsx 改为容器组件
export function MiniMap() {
  const containerRef = useRef<HTMLDivElement>(null)
  
  useEffect(() => {
    const minimap = new MinimapRenderer()
    minimap.init(containerRef.current)
    return () => minimap.destroy()
  }, [])
  
  return <div ref={containerRef} />
}
```

### 2026-03-12：装备 API 错误处理

**决策：** 客户端 + 服务端双重验证

**原因：**
1. 客户端未传递 characterId
2. 服务端未处理空 ID
3. 错误信息不明确
4. 影响装备面板显示

**实施：**
```typescript
// 客户端
const characterId = player?.characterId || 'default-id'
const response = await fetch(`/api/v1/equipment/${characterId}`)

// 服务端
if (!characterId || characterId.trim() === '') {
  return reply.code(400).send({ error: '角色 ID 不能为空' })
}
```
