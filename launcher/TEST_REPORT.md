## 🔧 修复的问题

### 问题 1: 路径嵌套

**问题描述：**
启动器会克隆到 `albion-lands/albion-lands/` 嵌套目录

**修复方案：**
```batch
:: 修复前
if not exist "albion-lands" (
    git clone ...
    cd albion-lands
)

:: 修复后
if not exist "server" (
    cd ..
    if not exist "albion-lands" (
        git clone ...
    )
    cd albion-lands
)
```

**验证：** ✅ 通过

---

### 问题 2: 进程清理不彻底

**问题描述：**
一次 taskkill 可能无法完全清理 Node 进程

**修复方案：**
```batch
:: 增强清理
taskkill /F /IM node.exe
timeout /t 3
taskkill /F /IM node.exe
timeout /t 2
```

**验证：** ✅ 通过
