🔄 呼噜大陆 - 强制清除缓存工具

⚠️ 重要：背包 UI 样式不生效是因为浏览器 CSS 缓存！请按以下步骤操作。

步骤 1: 关闭游戏页面

完全关闭所有 localhost:3001 的标签页

步骤 2: 清除浏览器缓存

Chrome/Edge:

按 Ctrl + Shift + Delete → 勾选"缓存的图像和文件" → 清除数据

或者使用开发者工具:

F12 → Network 标签 → 勾选"Disable cache" → 刷新页面

步骤 3: 拉取最新代码

cd F:\Tenbox\openclaw_1\03\albion-lands git pull origin main

步骤 4: 重启服务器

.\force-restart.bat

或者手动执行：

taskkill /F /IM node.exe cd client npm run dev

步骤 5: 强制刷新浏览器

访问 http://localhost:3001

按 Ctrl + Shift + R (强制刷新)

或者：

F12 → 右键刷新按钮 → 清空缓存并硬性重新加载

步骤 6: 验证修复

按 B 键 打开背包，应该看到：

  • ✅ 装备槽横向排列（3 列×2 行）
  • ✅ 文字水平显示（不是竖排）
  • ✅ 装备槽是正方形（不是竖长）
  • ✅ 物品栏格子是正方形（不是竖长）
  • ✅ 6 个装备槽（主手/副手/衣服/腿部/鞋子/饰品）
