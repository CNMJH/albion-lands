## 🔧 关键修复

### 1. Fastify 路由处理器签名
**问题**: 使用命名函数导致 `request.params` 为 undefined  
**修复**: 改用内联箭头函数  
**影响**: 5 个路由文件，17 个函数

### 2. 路由前缀重复
**问题**: `/api/v1/skills/skills/:id` 重复前缀  
**修复**: 移除路由文件内部的前缀  
**影响**: 技能查询、死亡记录

### 3. 路由冲突
**问题**: `/:id` 与 `/:characterId` 无法区分  
**修复**: 改为 `/by-character/:characterId`  
**影响**: 任务查询

### 4. UI 状态同步
**问题**: ChatUI 本地状态与 gameStore 不同步  
**修复**: 统一使用 `useGameStore().uiState`  
**影响**: 聊天框、所有 UI 面板

### 5. 数据库表缺失
**问题**: BankItem 表不存在  
**修复**: 创建脚本手动建表  
**影响**: 仓库查询
