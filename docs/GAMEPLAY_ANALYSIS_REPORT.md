## 🎮 实测问题记录

### 发现问题

1. **Canvas 焦点问题** ✅ 已修复
   - 现象：键盘无响应
   - 原因：Canvas 未自动 focus
   - 解决：添加 canvas.focus()

2. **装备 API 500 错误** ✅ 已修复
   - 现象：装备面板无法加载
   - 原因：characterId 验证失败
   - 解决：添加验证 + 从 gameStore 获取

3. **Vite 代理 500 错误** ✅ 已修复
   - 现象：部分 API 请求失败
   - 原因：Vite 代理配置问题
   - 解决：使用直接服务端 URL

4. **小地图冲突** ✅ 已修复
   - 现象：两个小地图实现
   - 原因：MiniMap.tsx 和 MinimapRenderer.ts 共存
   - 解决：删除 MiniMap.tsx

5. **技能特效缺失** ✅ 已修复
   - 现象：技能释放无视觉反馈
   - 原因：未连接 CombatRenderer
   - 解决：添加事件监听 + showSkillEffect()

6. **网络请求重复** ✅ 已修复
   - 现象：快速操作发送大量重复请求
   - 原因：未添加防抖机制
   - 解决：100ms 防抖窗口
