# 1. 游戏概述

## 1.1 游戏简介

《阿尔比恩大陆》是一款 2D 上帝视角的 MMORPG 浏览器游戏，灵感来源于 Albion Online。游戏核心特色：

- **玩家驱动的经济**：所有装备由玩家制造，所有资源由玩家采集
- **全物品掉落**：危险区域死亡掉落全部装备和背包物品
- **无职业限制**：装备决定能力，自由搭配
- **跨平台**：浏览器即可游玩，支持 AI 代理 (OpenClaw) 自动化

## 1.2 目标用户

| 用户类型 | 特征 | 需求 |
|----------|------|------|
| 休闲玩家 | 每天 1-2 小时 | 轻松采集、生产、社交 |
| 硬核玩家 | 每天 4+ 小时 | PVP、副本、世界 BOSS |
| 商人玩家 | 专注经济 | 市场交易、倒卖、垄断 |
| AI 玩家 | OpenClaw 代理 | 自动化采集、生产、挂机等 |

## 1.3 核心循环

```

│                     游戏核心循环                          │
├─────────────────────────────────────────────────────────┤

│      ↑                                              │   │
│      └────────────── 市场交易 ←─────────────────────┘   │
```

# 2. 世界观与背景

## 2.1 故事背景

在遥远的阿尔比恩大陆，古老的魔法能量滋养着这片土地。王国分裂为多个城邦，每个城邦由不同的贵族家族统治。

**核心设定：**
- 大陆被"魔力裂缝"分割成独立区域
- 城市由古代魔法保护，绝对安全
- 野外区域魔力不稳定，死亡会导致灵魂消散（装备掉落）
- 世界 BOSS 是古代魔物的化身，定期苏醒

## 2.2 阵营设定

| 阵营 | 主城 | 特色 | 加成 |
|------|------|------|------|
| 王国军 | 王城 | PVP 导向 | PVP 经验 +10% |
| 商人联盟 | 贸易港 | 经济导向 | 交易税 -5% |
| 猎人公会 | 猎鹰镇 | PVE 导向 | 采集速度 +10% |
| 法师议会 | 魔法塔 | 魔法导向 | 技能冷却 -5% |

玩家可选择初始阵营，但可自由前往任何区域。

## 2.3 区域安全等级

| 安全等级 | 颜色标识 | 死亡惩罚 | 怪物强度 | 资源品质 |
|----------|----------|----------|----------|----------|
| 安全区 (城市) | 🟢 绿色 | 无掉落 | 无怪物 | 基础 |
| 新手区 | 🔵 蓝色 | 无掉落 | 1-20 级 | 普通 |
| 普通野外 | 🟡 黄色 | 掉落 30% | 20-40 级 | 良好 |
| 危险区 | 🟠 橙色 | 掉落 50% | 40-60 级 | 稀有 |
| 极危区 | 🔴 红色 | 掉落 100% | 60-80 级 | 史诗 |
| 深渊区 | 🟣 紫色 | 掉落 100%+ 耐久损失 | 80 级 + | 传说 |

# 4. 角色与职业系统

## 4.1 角色属性

### 基础属性
| 属性 | 说明 | 影响 |
|------|------|------|
| 力量 (STR) | 物理攻击力 | 近战伤害、负重 |
| 敏捷 (AGI) | 速度与暴击 | 攻击速度、闪避、暴击率 |
| 智力 (INT) | 魔法能力 | 法术伤害、魔法值、冷却缩减 |
| 体质 (CON) | 生存能力 | 生命值、防御、回复 |
| 感知 (PER) | 特殊能力 | 采集效率、暴击伤害、掉落率 |

### 衍生属性
- **生命值 (HP)** = 基础值 + CON × 10
- **魔法值 (MP)** = 基础值 + INT × 8
- **物理攻击** = 武器基础 + STR × 0.5
- **法术攻击** = 武器基础 + INT × 0.8
- **物理防御** = 装备基础 + CON × 0.3
- **暴击率** = AGI × 0.1%
- **闪避率** = AGI × 0.05%

## 4.2 职业系统（无限制）

游戏**无固定职业**，由装备和技能决定角色定位：

| 定位 | 武器 | 防具 | 属性侧重 | 玩法 |
|------|------|------|----------|------|
| 战士 | 剑、斧、锤 | 板甲 | STR+CON | 近战坦克/DPS |
| 弓箭手 | 弓、弩 | 皮甲 | AGI+STR | 远程物理 DPS |
| 法师 | 法杖、魔典 | 布甲 | INT+CON | 远程法术 DPS |
| 刺客 | 匕首、短剑 | 皮甲 | AGI+STR | 近战爆发 DPS |
| 牧师 | 圣杖、圣典 | 布甲/板甲 | INT+CON | 治疗/辅助 |
| 猎人 | 长矛、陷阱 | 皮甲 | AGI+PER | PVE  specialist |

## 4.3 技能系统

### 技能类型
| 类型 | 说明 | 示例 |
|------|------|------|
| 主动技能 | 玩家主动释放 | 火球术、冲锋、治疗术 |
| 被动技能 | 永久生效 | 增加攻击、增加防御 |
| 武器技能 | 绑定武器类型 | 剑术、弓术、法术 |
| 生活技能 | 采集/制造 | 采矿、伐木、锻造 |

### 技能槽位
- **武器技能槽：** 3 个（由武器决定）
- **通用技能槽：** 2 个（可自由装备）
- **被动技能槽：** 3 个

### 技能获取
- 武器技能：装备对应武器自动解锁
- 通用技能：技能书学习（副本掉落/市场购买）
- 生活技能：NPC 导师学习

## 4.4 装备系统

### 装备品质
| 品质 | 颜色 | 掉率 | 属性倍率 | 来源 |
|------|------|------|----------|------|
| 普通 | ⚪ 白色 | 60% | 100% | 基础制造 |
| 精良 | 🟢 绿色 | 25% | 120% | 进阶制造 |
| 稀有 | 🔵 蓝色 | 10% | 150% | 副本/BOSS |
| 史诗 | 🟣 紫色 | 4% | 200% | 世界 BOSS/活动 |
| 传说 | 🟠 橙色 | 1% | 300% | 稀有掉落/成就 |

### 装备部位
```
头部 → 头盔/帽子
颈部 → 项链
胸部 → 盔甲/长袍
手部 → 手套
腿部 → 裤子/裙子
脚部 → 靴子
主手 → 武器/盾牌
副手 → 武器/饰品
戒指 ×2
```

### 装备耐久
- 所有装备有耐久度（100/100）
- 死亡时耐久损失 10-20%
- 耐久≤0 时装备失效
- 可在城市铁匠铺修复

## 7.2 物品价值体系

### 基础公式
```
物品基础价值 = 材料成本 + 制造时间成本 + 稀有度系数

材料成本 = Σ(材料数量 × 材料单价)
制造时间成本 = 制造时间 (分钟) × 每分钟金币价值
稀有度系数：普通 1.0 / 精良 1.5 / 稀有 2.0 / 史诗 3.0 / 传说 5.0
```

### 市场定价机制
```
市场价格 = 基础价值 × 供需系数

供需系数 = 需求订单量 / 供应订单量
- 系数 > 1.5：价格上涨
- 系数 < 0.7：价格下跌
- 系统每日自动调整参考底价
```

## 8.2 伤害计算

### 物理伤害公式
```
最终伤害 = (攻击力 - 防御力) × 技能系数 × 暴击系数 × 随机浮动

攻击力 = 武器基础攻击 + 属性加成
防御力 = 装备防御 × (1 - 穿透率)
技能系数 = 技能基础倍率 + 技能等级加成
暴击系数 = 1 + (暴击伤害 - 100)%
随机浮动 = 0.9 ~ 1.1
```

### 法术伤害公式
```
最终伤害 = 法术攻击 × 技能系数 × 抗性系数 × 随机浮动

抗性系数 = 1 - (目标抗性 / (目标抗性 + 100))
```

```typescript
// WebSocket 连接管理
class NetworkManager {
  private ws: WebSocket | null = null;
  private reconnectAttempts = 0;
  private maxReconnectAttempts = 5;
  
  connect(url: string): Promise<void> {
    return new Promise((resolve, reject) => {
      this.ws = new WebSocket(url);
      
      this.ws.onopen = () => {
        this.reconnectAttempts = 0;
        resolve();
      };
      
      this.ws.onmessage = (event) => {
        this.handleMessage(event.data);
      };
      
      this.ws.onclose = () => {
        this.handleReconnect();
      };
    });
  }
  
  send(packet: GamePacket): void {
    if (this.ws?.readyState === WebSocket.OPEN) {
      this.ws.send(this.encode(packet));
    }
  }
  
  private handleReconnect(): void {
    if (this.reconnectAttempts < this.maxReconnectAttempts) {
      this.reconnectAttempts++;
      setTimeout(() => this.connect(this.serverUrl), 1000 * this.reconnectAttempts);
    }
  }
}
```

```typescript
class AssetLoader {
  private cache: Map<string, any> = new Map();
  private loadingQueue: string[] = [];
  
  async load<T>(key: string, loader: () => Promise<T>): Promise<T> {
    // 检查缓存
    if (this.cache.has(key)) {
      return this.cache.get(key);
    }
    
    // 加入加载队列
    this.loadingQueue.push(key);
    
    try {
      const asset = await loader();
      this.cache.set(key, asset);
      return asset;
    } finally {
      this.loadingQueue = this.loadingQueue.filter(k => k !== key);
    }
  }
  
  // 预加载区域资源
  async preloadZone(zoneId: string): Promise<void> {
    const zoneConfig = await this.loadZoneConfig(zoneId);
    
    // 预加载地图图块
    await Promise.all(
      zoneConfig.tilesets.map(tileset => 
        this.load(`tileset:${tileset}`, () => loadTexture(tileset))
      )
    );
    
    // 预加载怪物精灵
    await Promise.all(
      zoneConfig.monsters.map(monster =>
        this.load(`monster:${monster.id}`, () => loadSpriteSheet(monster.sprite))
      )
    );
  }
}
```

```typescript
class GameServer {

  zones: Map<string, Zone>;
  
  // 玩家管理
  players: Map<string, Player>;
  
  // 更新循环
  startGameLoop(): void {
    const TICK_RATE = 30; // 30 TPS
    setInterval(() => this.tick(), 1000 / TICK_RATE);
  }
  
  tick(): void {
    // 更新所有玩家
    this.players.forEach(player => player.update());
    
    // 更新所有怪物
    this.zones.forEach(zone => zone.updateMonsters());
    
    // 碰撞检测
    this.checkCollisions();
    
    // 状态同步
    this.syncState();
  }
  
  // 战斗计算
  calculateDamage(attacker: Player, target: Entity): DamageResult {
    // 伤害公式实现
    const baseDamage = attacker.attack - target.defense;
    const critMultiplier = this.rollCrit(attacker.critRate);
    const randomFactor = 0.9 + Math.random() * 0.2;
    
    return {
      damage: Math.floor(baseDamage * critMultiplier * randomFactor),
      isCrit: critMultiplier > 1,
    };
  }
}
```

# 13. OpenClaw 集成方案

## 13.1 OpenClaw 简介

OpenClaw 是一个开源的游戏自动化框架，允许玩家创建脚本来自动化游戏操作。

**本项目集成目标：**
- 支持 OpenClaw 脚本接入游戏
- 提供有限的 API 供脚本调用
- 保证 AI 玩家与真人玩家平衡

## 13.2 API 设计原则

### 设计原则
1. **有限性**：只开放必要的 API，不提供游戏内挂功能
2. **延迟性**：AI 操作添加人为延迟，模拟真人操作
3. **可见性**：AI 玩家有标识，其他玩家可识别
4. **限制性**：AI 玩家收益有限制，防止刷资源

### API 分类
| 类别 | 开放程度 | 说明 |
|------|----------|------|
| 移动 API | ✅ 开放 | 角色移动、寻路 |
| 交互 API | ✅ 开放 | NPC 对话、物品使用 |
| 战斗 API | ⚠️ 限制 | 添加延迟、冷却 |
| 交易 API | ⚠️ 限制 | 价格限制、频率限制 |
| 聊天 API | ❌ 禁止 | 防止垃圾信息 |
| 地图 API | ⚠️ 限制 | 部分信息隐藏 |

```typescript
// OpenClaw API 服务端实现
class OpenClawService {
  // API 调用延迟（模拟真人）
  private readonly ACTION_DELAY = 200;  // 基础延迟 ms
  private readonly COMBAT_DELAY = 500;  // 战斗延迟 ms
  
  // 速率限制
  private rateLimiter: RateLimiter;
  
  constructor() {
    this.rateLimiter = new RateLimiter({
      actionsPerMinute: 60,    // 每分钟最多 60 个动作
      combatActionsPerMinute: 30, // 战斗动作每分钟 30 次
      tradeActionsPerHour: 100,   // 交易每小时 100 次
    });
  }
  
  // 移动 API 实现
  async moveTo(
    playerId: string, 
    x: number, 
    y: number
  ): Promise<void> {
    // 速率限制检查
    await this.rateLimiter.checkLimit(playerId, 'move');
    
    // 添加人为延迟
    await this.sleep(this.ACTION_DELAY + Math.random() * 100);
    
    // 验证目标位置
    const player = await this.getPlayer(playerId);
    const valid = await this.validateMoveTarget(player, x, y);
    
    if (!valid) {
      throw new Error('Invalid move target');
    }
    
    // 执行移动
    await this.gameServer.movePlayer(playerId, x, y);
  }
  
  // 战斗 API 实现
  async attackTarget(
    playerId: string, 
    targetId: string
  ): Promise<void> {
    // 战斗速率限制
    await this.rateLimiter.checkLimit(playerId, 'combat');
    
    // 战斗延迟（模拟真人反应）
    await this.sleep(this.COMBAT_DELAY + Math.random() * 300);
    
    // 验证目标
    const target = await this.getEntity(targetId);
    if (!target || target.isPlayer) {
      // PVP 额外限制
      await this.validatePVP(playerId, targetId);
    }
    
    // 执行攻击
    await this.gameServer.attack(playerId, targetId);
  }
  
  // 市场 API 实现
  async createOrder(
    playerId: string, 
    order: OrderRequest
  ): Promise<OrderResult> {
    // 交易速率限制
    await this.rateLimiter.checkLimit(playerId, 'trade');
    
    // 价格验证（防止恶意操纵）
    const marketPrice = await this.getMarketPrice(order.itemId);
    if (order.price > marketPrice * 2 || order.price < marketPrice * 0.5) {
      throw new Error('Price too far from market value');
    }
    
    // 创建订单
    return await this.marketService.createOrder(playerId, order);
  }
  
  private sleep(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}
```

## 14.1 平衡性设计

```typescript
class AntiCheatSystem {
  // 移动速度检测
  checkMoveSpeed(playerId: string, from: Vector2D, to: Vector2D, time: number): void {
    const distance = this.calculateDistance(from, to);
    const speed = distance / time;
    
    const player = this.getPlayer(playerId);
    const maxSpeed = player.getMoveSpeed() * 1.2; // 20% 容错
    
    if (speed > maxSpeed) {
      this.logViolation(playerId, 'speed_hack', { speed, maxSpeed });
      this.punish(playerId, 'speed_hack');
    }
  }
  
  // 操作模式分析（检测脚本）
  analyzeBehavior(playerId: string): BehaviorScore {
    const actions = this.getRecentActions(playerId, 100);
    
    // 检测规律性（脚本通常有固定模式）
    const regularity = this.calculateRegularity(actions);
    
    // 检测反应时间分布
    const reactionTimes = this.getReactionTimes(actions);
    const reactionVariance = this.calculateVariance(reactionTimes);
    
    // 真人玩家反应时间方差大，脚本方差小
    if (regularity > 0.9 && reactionVariance < 0.1) {
      return { score: 0.2, reason: '疑似脚本' };
    }
    
    return { score: 1.0, reason: '正常' };
  }
  
  // 处罚措施
  async punish(playerId: string, violation: string): Promise<void> {
    const player = await this.getPlayer(playerId);
    const violations = await this.getViolationHistory(playerId);
    
    if (violations.length >= 3) {
      // 多次违规，封禁
      await this.banPlayer(playerId, 7); // 封禁 7 天
    } else if (violations.length >= 1) {
      // 首次违规，警告 + 限制
      await this.warnPlayer(playerId);
      await this.applyRestriction(playerId, {
        expRate: 0.5,
        dropRate: 0.5,
        duration: 24 * 3600 * 1000, // 24 小时
      });
    }
  }
}
```

```typescript
class AlertSystem {

    const threshold = this.getThreshold(metric);

      const alert: Alert = {
        playerId,
        metric,
```
