## 🎨 设计亮点

### 1. 颜色系统统一

```css
:root {
  --primary-color: #3498db;      /* 主色调 */
  --accent-color: #e74c3c;       /* 强调色 */
  --success-color: #27ae60;      /* 成功色 */
  --warning-color: #f39c12;      /* 警告色 */
  --bg-dark: #16213e;            /* 深色背景 */
  --text-primary: #ffffff;       /* 主文字 */
}
```

### 2. 毛玻璃效果

```css
backdrop-filter: blur(10px);
background: rgba(22, 33, 62, 0.95);
```

### 3. 多层阴影

```css
box-shadow: 
  0 4px 16px rgba(0, 0, 0, 0.5),           /* 外阴影 */
  inset 0 1px 0 rgba(255, 255, 255, 0.1);  /* 内高光 */
```

### 4. 渐变效果

```css
/* 按钮渐变 */
background: linear-gradient(180deg, #3498db 0%, #2980b9 100%);

/* 血条渐变 */
background: linear-gradient(90deg, #e74c3c 0%, #c0392b 50%, #e74c3c 100%);
```

### 5. 动画效果

```css
/* 窗口滑入 */
@keyframes windowSlideIn {
  from { opacity: 0; transform: translate(-50%, -45%); }
  to { opacity: 1; transform: translate(-50%, -50%); }
}

/* 传奇装备发光 */
@keyframes legendaryGlow {
  0%, 100% { box-shadow: 0 0 12px rgba(243, 156, 18, 0.6); }
  50% { box-shadow: 0 0 20px rgba(243, 156, 18, 0.8); }
}
```

## 🔧 使用指南

### 1. 应用优化样式

```css
/* 在组件文件中引用 */
import './UIOverlay-optimized.css'
import './CharacterInfo-optimized.css'
import './Inventory-optimized.css'
import './SkillBar-optimized.css'
import './ChatBox-optimized.css'
```

### 2. 使用 CSS 变量

```css
.my-component {
  background: var(--bg-dark);
  border: 2px solid var(--border-color);
  color: var(--text-primary);
  padding: var(--spacing-lg);
  border-radius: var(--radius-lg);
}
```

### 3. 使用通用类

```html
<!-- 面板 -->
<div class="game-panel">
  <div class="game-panel-header">
    <span class="game-panel-title">标题</span>
  </div>
  <div class="game-panel-content">内容</div>
</div>

<!-- 按钮 -->
<button class="game-button">主要</button>
<button class="game-button secondary">次要</button>
<button class="game-button danger">危险</button>
```
