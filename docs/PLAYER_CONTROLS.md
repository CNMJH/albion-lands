# 玩家操作指南

## 基础操作

### 移动
- **WASD** 或 **方向键 ↑↓←→** - 键盘移动
- **鼠标右键** - 点击地面移动到目标位置

### 战斗
- **鼠标左键** - 普通攻击
- **空格键** - 普通攻击（快捷键）
- **数字键 1-8** - 使用技能快捷栏

### 交互
- **E 键** - 与附近对象交互
  - 采集资源（矿石、树木等）
  - 与 NPC 对话
  - 拾取物品

### UI 快捷键
| 按键 | 功能 | 说明 |
|------|------|------|
| **B** | 背包 | 打开/关闭背包界面 |
| **C** | 制作 | 打开/关闭制作界面 |
| **Q** | 任务 | 打开/关闭任务界面 |
| **F** | 好友 | 打开/关闭好友界面 |
| **Enter** | 聊天 | 打开/关闭聊天输入框 |
| **Esc** | 关闭 UI | 关闭所有打开的界面（待实现） |

## 控制方案说明

### 移动系统
- 移动速度：**200 像素/秒**
- 支持 **8 方向移动**（包括对角线）
- 对角线移动已 **归一化**（速度一致）
- 移动更新频率：**50ms**（每秒 20 次）

### 战斗系统
- 攻击冷却时间：**1 秒**
- 攻击范围：**50 像素**
- 支持 **键盘 + 鼠标** 双模式

### 交互系统
- 交互范围：**60 像素**
- 自动检测最近的交互对象

## 技术实现

### 玩家操作系统 (PlayerControlsSystem)
统一管理所有玩家输入，包括：
- 键盘输入监听
- 鼠标输入监听
- 移动处理
- 攻击处理
- 交互处理
- 技能使用
- UI 快捷键

### 文件位置
- `client/src/systems/PlayerControlsSystem.ts` - 玩家操作系统核心
- `client/src/renderer/GameCanvas.tsx` - 集成到游戏循环
- `client/src/hooks/useUIManager.ts` - UI 状态管理 Hook

### 事件系统
玩家操作系统触发以下事件：
- `playerAttack` - 玩家攻击事件
- `playerInteract` - 玩家交互事件
- `playerSkill` - 玩家技能事件
- `uiToggle` - UI 切换事件

## 测试方法

1. **移动测试**
   - 使用 WASD 移动，观察玩家蓝色圆形是否移动
   - 使用鼠标右键点击地面，玩家应移动到目标位置
   - 测试对角线移动速度是否一致

2. **攻击测试**
   - 鼠标左键点击，观察控制台是否输出攻击日志
   - 按空格键，应触发相同攻击效果
   - 连续点击测试攻击冷却（1 秒）

3. **交互测试**
   - 靠近资源点或 NPC
   - 按 E 键，观察控制台是否输出交互日志

4. **UI 快捷键测试**
   - 按 B 键，背包界面应打开/关闭
   - 按 C 键，制作界面应打开/关闭
   - 按 Q 键，任务界面应打开/关闭
   - 按 F 键，好友界面应打开/关闭
   - 按 Enter 键，聊天框应激活

5. **技能测试**
   - 按数字键 1-8，观察控制台是否输出技能使用日志

## 注意事项

### 聊天框激活时
- 当聊天框激活时，游戏快捷键（WASD、E、B 等）会被禁用
- 避免在聊天时意外触发游戏操作

### UI 互斥
- 同一时间只能打开一个 UI 面板
- 打开新面板会自动关闭当前面板
- 再次按下相同快捷键会关闭当前面板

### 网络同步
- 所有操作都会发送到服务端进行验证
- 移动、攻击、交互等操作都有网络消息
- 服务端会返回确认消息更新客户端状态
