### 3. 定时器管理优化 - CraftingUI.tsx ✅
**文件**: `client/src/components/CraftingUI.tsx`
**问题**: setInterval 在组件卸载时可能未清理
**风险**: 内存泄漏，状态更新到已卸载组件

**修复前**:
```typescript
const interval = setInterval(() => {
  setCraftProgress(prev => {
    if (prev >= 100) {
      clearInterval(interval)
      return 100
    }
    return prev + 10
  })
}, recipe.craftTime / 10)
```

**修复后**:
```typescript
// 添加 ref 存储定时器
const craftIntervalRef = React.useRef<number | null>(null)

// 使用 ref 存储
craftIntervalRef.current = window.setInterval(() => {
  setCraftProgress(prev => {
    if (prev >= 100) {
      if (craftIntervalRef.current) {
        clearInterval(craftIntervalRef.current)
        craftIntervalRef.current = null
      }
      return 100
    }
    return prev + 10
  })
}, recipe.craftTime / 10)

// useEffect 清理
useEffect(() => {
  return () => {
    if (craftIntervalRef.current) {
      clearInterval(craftIntervalRef.current)
    }
  }
}, [])
```

**影响**: 防止内存泄漏和状态更新错误
