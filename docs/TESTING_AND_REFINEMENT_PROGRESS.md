### 2. 修复的问题

#### 问题 1: @fastify/static 版本不兼容
**错误:**
```
fastify-plugin: @fastify/static - expected '5.x' fastify version, '4.29.1' is installed
```

**解决:**
```bash
npm install @fastify/static@^7.0.0
```

**文件:** `server/package.json`

#### 问题 2: 路由导入错误
**错误:**
```
Plugin must be a function or a promise. Received: 'undefined'
```

**原因:** gm.ts 和 social.ts 使用命名导出而非默认导出

**解决:**
```typescript
// server/src/routes/index.ts
import { gmRoutes } from './gm'
import { socialRoutes } from './social'
```

#### 问题 3: 健康检查路由错误
**错误:** 测试脚本访问 `/api/health` 但实际路由是 `/health`

**解决:** 修改测试脚本使用正确的路由

#### 问题 4: add-world-horn.js 字段错误
**错误:**
```
Unknown argument `description`
```

**原因:** Item 模型没有 description 字段

**解决:** 移除 description 字段

### 问题 1: 服务端后台启动失败

**现象:** 使用 `&` 后台启动时，服务端立即关闭

**原因:** 
- Shell 脚本后台进程管理问题
- 父进程退出导致子进程关闭

**临时解决:**
- 使用 `npm run dev` 前台启动

**建议:** 使用 pm2 管理服务端进程
```bash
npm install -g pm2
pm2 start npm --name "hulu-server" -- run dev
pm2 logs hulu-server
```

### 2. 测试框架
**建议:** 使用 Jest 或 Vitest
```bash
npm install -D jest @types/jest
```

### 3. CI/CD
**建议:** 添加 GitHub Actions
```yaml
name: Test
on: [push]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - run: npm install
      - run: npm test
```

### 4. 测试数据库
**建议:** 使用独立测试数据库
