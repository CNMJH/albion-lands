# 呼噜大陆 - 美术资源需求文档

## 📋 概述

本文档列出《呼噜大陆》游戏所需的美术资源清单和风格指南。

## 🎨 美术风格

- **视角：** 2D 上帝视角（45 度斜视角）
- **风格：** 像素艺术 + 可爱卡通风格
- **色调：** 明亮、鲜艳、奇幻风格
- **参考：** 阿尔比恩 Online、星露谷物语、泰拉瑞亚

---

## 👤 角色设计

### 1. 玩家角色

**基础职业：**

| 职业 | 描述 | 武器 | 防具 |
|------|------|------|------|
| 战士 | 近战物理输出 | 剑/斧 | 重甲 |
| 法师 | 远程魔法输出 | 法杖 | 布甲 |
| 游侠 | 远程物理输出 | 弓 | 皮甲 |
| 牧师 | 治疗辅助 | 权杖 | 布甲 |

**角色状态：**
- 站立（4 方向）
- 行走（4 方向）
- 跑步（4 方向）
- 攻击（4 方向）
- 受击
- 死亡
- 采集动作

**尺寸：** 32x32 像素 或 64x64 像素

### 2. NPC

| 类型 | 描述 | 数量 |
|------|------|------|
| 新手引导 NPC | 友好村民 | 3-5 |
| 商人 | 武器商、防具商、药水商 | 5-8 |
| 铁匠 | 装备强化 | 1-2 |
| 银行家 | 仓库管理 | 1-2 |
| 公会管理员 | 公会相关 | 1-2 |
| 任务 NPC | 任务发布 | 10+ |

### 3. 怪物

**新手区（1-10 级）：**
- 史莱姆（绿色、蓝色、红色）
- 哥布林（普通、弓箭手、法师）
- 野狼
- 蜘蛛

**中级区（10-30 级）：**
- 骷髅兵
- 兽人
- 蝙蝠
- 毒蝎

**高级区（30-50 级）：**
- 恶魔
- 巨龙
- 亡灵骑士
- 元素生物

**BOSS：**
- 区域 BOSS（每区域 1-2 个）
- 世界 BOSS（全服事件）

## 📝 美术规格

### 角色/怪物

- **尺寸：** 32x32 或 64x64 像素
- **格式：** PNG（透明背景）
- **帧数：** 每个动作 4-8 帧
- **方向：** 4 方向或 8 方向

### 场景

- **Tile 尺寸：** 32x32 像素
- **格式：** PNG
- **Tileset 大小：** 根据需求（如 512x512）

### 物品图标

- **尺寸：** 32x32 或 48x48 像素
- **格式：** PNG（透明背景）

### UI

- **分辨率：** 适配 1920x1080
- **格式：** PNG
- **9 宫格：** 用于可缩放背景

## 🎯 优先级

### 第一阶段（核心框架测试）

1. ✅ 玩家角色（1 个基础职业）
2. ✅ 基础动作（站立、行走）
3. ✅ 新手村场景
4. ✅ 基础 UI 界面
5. ✅ 简单怪物（史莱姆）

### 第二阶段（核心玩法）

1. 4 个基础职业
2. 完整动作集合
3. 3-5 种怪物
4. 物品图标（50+）
5. 技能特效（10+）

### 第三阶段（多人联机）

1. 更多怪物
2. 完整 UI 套装
3. 9 大区域场景
4. NPC 角色

### 第四阶段（内容扩展）

1. BOSS 设计
2. 特殊效果
3. 时装系统
4. 坐骑系统
