## 🎮 操作指南

### 打开/关闭装备面板

**方法 1：快捷键**
```
按 C 键 → 打开装备面板
再按 C 键 → 关闭装备面板
```

**方法 2：点击按钮**
```
点击右下角 "⚔️ 装备 (C)" 按钮
```

### 装备物品

**步骤 1：打开装备面板**
```
按 C 键打开装备界面
```

**步骤 2：选择槽位**
```
点击空槽位（如"主手"）
→ 弹出物品选择窗口
```

**步骤 3：选择物品**
```
在物品列表中选择要装备的物品
→ 点击"装备"按钮
```

**步骤 4：确认装备**
```
槽位显示已装备物品
→ 绿色边框表示已装备
→ 右侧显示装备名称和等级
```

### 卸下装备

**方法 1：点击槽位**
```
点击已装备的槽位
→ 装备自动卸下
→ 物品返回背包
```

**方法 2：右键菜单**
```
右键点击已装备物品
→ 选择"卸下"
```

## 📊 属性说明

### 基础属性（5 种）

| 属性 | 图标 | 说明 | 影响 |
|------|------|------|------|
| **攻击力** | ⚔️ | 物理/魔法攻击 | 技能伤害、普攻伤害 |
| **防御力** | 🛡️ | 物理/魔法防御 | 减少受到的伤害 |
| **生命值** | ❤️ | HP 最大值 | 生存能力、血量上限 |
| **攻击速度** | ⚡ | 攻击频率 | DPS、技能冷却 |
| **移动速度** | 👟 | 移动快慢 | 风筝、追击、跑图 |

## ⚙️ 战力评分

### 计算公式

```javascript
总评分 = 攻击力 × 2 
       + 防御力 × 1.5 
       + 生命值 × 0.5 
       + 攻击速度 × 10 
       + 移动速度 × 0.1
```

### 评分示例

| 评分 | 等级 | 说明 |
|------|------|------|
| < 100 | 新手 | 初始装备 |
| 100-300 | 普通 | T1-T2 装备 |
| 300-600 | 精英 | T3-T4 装备 |
| 600-1000 | 大师 | T5 装备 |
| > 1000 | 传奇 | 顶级装备 |

## 🔧 技术细节

### 组件结构

```
EquipmentPanel (主组件)
├── equipment-toggle-btn (切换按钮)
├── equipment-panel-window (主窗口)
│   ├── equipment-header (头部)
│   ├── equipment-content (内容区)
│   │   ├── equipment-slots-section (装备槽位区)
│   │   │   └── equipment-slot-item × 6 (槽位物品)
│   │   └── character-stats-section (属性区)
│   │       ├── stats-list (属性列表)
│   │       │   └── stat-row × 5 (属性行)
│   │       └── power-score (战力评分)
│   └── item-selection-modal (物品选择弹窗)
│       └── items-grid (物品网格)
│           └── item-card (物品卡片)
```

### 状态管理

```typescript
interface EquipmentPanelState {
  isOpen: boolean              // 面板是否打开
  characterId: string          // 角色 ID
  equipment: Equipment         // 装备数据
  stats: CharacterStats        // 属性数据
  selectedSlot: EquipmentSlot | null  // 选中的槽位
}
```

### API 调用

```typescript
// 加载装备
GET /api/v1/equipment/:characterId

// 加载属性
GET /api/v1/equipment/:characterId/stats

// 装备物品
POST /api/v1/equipment/:characterId/equip
Body: { itemId: string, slot: EquipmentSlot }

// 卸下装备
POST /api/v1/equipment/:characterId/unequip
Body: { slot: EquipmentSlot }
```

## 🐛 常见问题

### Q1: 点击槽位没有反应？

**检查项：**
- [ ] 确认角色 ID 已加载
- [ ] 确认 EquipmentSystem 已初始化
- [ ] 检查控制台是否有错误

### Q2: 装备后属性没有更新？

**解决方法：**
```typescript
// 手动刷新属性
await equipmentSystem.loadStats()
```

### Q3: 物品选择窗口显示空？

**原因：** 背包中没有对应槽位的可装备物品

**解决方法：**
1. 打开背包（B 键）
2. 获取对应类型的装备
3. 再次打开装备面板

### Q4: 战力评分计算不准确？

**检查项：**
- [ ] 确认所有装备已正确加载
- [ ] 确认物品属性数据完整
- [ ] 检查计算公式是否正确
