## 🛠️ 技术栈

### 客户端
- **渲染引擎**: Pixi.js v7
- **UI 框架**: React 18 + TypeScript
- **状态管理**: Zustand
- **构建工具**: Vite
- **端口**: 3001

### 服务端
- **运行时**: Node.js 18+
- **Web 框架**: Fastify
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **ORM**: Prisma
- **通信**: WebSocket + HTTP API
- **端口**: 3002

### 开发工具
- **测试**: Playwright + Shell 脚本
- **版本控制**: Git
- **包管理**: npm

## 🚀 快速开始

### Windows 用户
```batch
# 一键启动
launcher.bat
```

### Linux/Mac 用户
```bash
# 启动服务端
cd server && npm run dev

# 启动客户端（新终端）
cd client && npm run dev

# 验证 API
./scripts/verify-all.sh
```

- **客户端**: http://localhost:3001
- **服务端**: http://localhost:3002

## 🎯 操作指南

### 基础操作
- **移动**: 鼠标右键点击地面
- **攻击**: 鼠标右键点击敌人
- **技能**: QWER 键
- **拾取**: E 键（80px 范围内）
- **冲刺**: Shift 键（1.5 倍速度）

### UI 快捷键
- **B**: 打开/关闭背包
- **C**: 打开/关闭装备
- **M**: 打开/关闭拍卖行
- **F1**: 死亡统计面板
- **F2**: 复活点绑定

### 聊天
- **Enter**: 打开聊天框
- **区域聊天**: 默认免费
- **全服聊天**: 消耗大喇叭×1
