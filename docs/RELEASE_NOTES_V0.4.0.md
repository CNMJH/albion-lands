## 🎮 游戏特性总览

### 核心玩法
- ⚔️ **战斗系统**: LOL 风格右键移动/攻击，QWERAS 技能
- 🎒 **背包系统**: 6 槽位装备，正方形格子，拖拽操作
- 💰 **经济系统**: 自由交易，拍卖行，采集制造
- ☠️ **死亡掉落**: 安全区/危险区差异化掉落
- ⚡ **PVP 系统**: 安全区保护，伤害计算，击杀公告
- 🤝 **交易系统**: 原子交换，双重确认，距离限制
- 📊 **排行榜**: 等级/财富/PVP/PVE 多维度排名

### 游戏内容
- 🗺️ **地图系统**: 5 区域，安全等级机制，网格地面
- 👾 **怪物系统**: 17+ 种怪物，AI 行为，自动生成
- 📜 **任务系统**: 每日任务，成就系统，进度追踪
- 🏪 **NPC 系统**: 商人/任务/服务/守卫/黑市
- 🎭 **角色系统**: 装备/属性/技能/死亡统计/复活点

### 用户体验
- 🎨 **UI 优化**: 深色主题，毛玻璃效果，现代化设计
- 🔔 **通知系统**: Toast 消息，4 种类型，自动消失
- ⌨️ **快捷键**: F1-F12 功能键，WASD 移动，ESC 关闭 UI
- 📱 **响应式**: 适配不同屏幕尺寸
- 🎵 **音频管理**: BGM 淡入淡出，SFX 并发播放
- 📸 **截图功能**: PrintScreen 快捷键，PNG 格式保存

### 开发者工具
- 🔍 **调试控制台**: F12 打开，12 个内置命令
- 📈 **性能监控**: F11 打开，FPS/内存/延迟显示
- 🎯 **调试覆盖层**: 实时显示玩家位置/移动日志
- 🧪 **测试工具**: 浏览器测试页面，网络抓包工具

## 🚀 快速开始

### Windows 用户
```batch
# 1. 下载安装包或克隆仓库
git clone https://github.com/CNMJH/albion-lands.git

# 2. 一键安装
install.bat

# 3. 启动游戏
start.bat
```

### 手动安装
```bash
# 安装依赖
cd server && npm install
cd ../client && npm install

# 启动服务
cd server && npm run dev
cd client && npm run dev

# 访问游戏
http://localhost:3001
```
