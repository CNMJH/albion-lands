# 🎮 H5 网页端 GM 工具

## 📋 概述

集成到游戏客户端的 H5 网页端 GM 管理工具，可以在游戏内直接打开使用。

**访问方式**: 游戏内按 `Ctrl+G` 或点击 GM 按钮

---

## 🎯 功能特性

### 1. 📊 仪表盘
- 服务器统计数据
- 实时日志查看

### 2. 👥 玩家管理
- 查看玩家列表
- 修改等级
- 给予物品
- 给予货币
- 传送玩家

### 3. 🎒 物品管理
- 查看所有物品
- 稀有度显示
- 物品信息

### 4. 🌲 资源监控
- 资源节点状态
- 采集次数显示
- 位置信息

### 5. 📝 日志查看
- 按类型筛选
- 时间排序
- 详细消息

### 6. 🖥️ 服务器状态
- 运行时间
- 内存使用
- 系统信息

## 🎮 使用方法

### 打开 GM 面板
1. 在游戏内按 `Ctrl+G`
2. 或点击 GM 按钮（如果添加）

### 关闭 GM 面板
1. 再次按 `Ctrl+G`
2. 或点击右上角 `×` 按钮
3. 或点击面板外部区域

## 🔧 功能详解

### 仪表盘
显示服务器实时数据：
- **总玩家数**: 注册角色总数
- **在线玩家**: 当前在线人数
- **物品总数**: 物品模板数量
- **市场订单**: 拍卖行订单数
- **最近日志**: 最新 10 条日志

### 玩家管理

#### 修改等级
1. 点击玩家列表中的"改等级"按钮
2. 输入新等级 (1-100)
3. 确认后即可生效

#### 给予物品
1. 点击"给物品"按钮
2. 输入物品 ID (如 `iron_sword`)
3. 输入数量
4. 确认给予

#### 给予货币
1. 点击"给货币"按钮
2. 输入银币数量
3. 输入金币数量
4. 确认给予

#### 传送玩家
1. 点击"传送"按钮
2. 输入区域 ID (如 `zone_1`)
3. 输入 X 坐标
4. 输入 Y 坐标
5. 确认传送

### 物品管理
查看所有物品模板：
- 物品名称
- 稀有度 (颜色标识)
- 类型
- 需要等级
- 基础价格

稀有度颜色：
- ⚪ 白色 - Common
- 🟢 绿色 - Uncommon
- 🔵 蓝色 - Rare
- 🟣 紫色 - Epic
- 🟠 橙色 - Legendary

### 资源监控
查看当前所有资源节点：
- 资源名称
- 资源类型
- 所在区域
- 剩余采集次数
- 坐标位置

### 日志查看
筛选和查看游戏日志：
- 全部类型
- GM 操作
- GM 广播
- 战斗日志
- 交易日志

### 服务器状态
查看服务器运行状态：
- 运行时间 (时：分)
- 内存使用 (MB)
- 操作系统
- Node.js 版本

## 📡 API 接口

GM 面板使用的 API 接口：

### 仪表盘
```
GET /api/v1/gm/dashboard
```

### 玩家
```
GET  /api/v1/gm/players
POST /api/v1/gm/players/:id/level
POST /api/v1/gm/players/:id/items
POST /api/v1/gm/players/:id/currency
POST /api/v1/gm/players/:id/teleport
```

### 物品
```
GET /api/v1/gm/items
```

### 资源
```
GET /api/v1/gm/resources
```

### 日志
```
GET /api/v1/gm/logs?type=&limit=
```

### 服务器
```
GET /api/v1/gm/server/status
```

## 🛠️ 开发说明

### 添加新功能

1. **在组件中添加标签页**
```tsx
<button 
  className={`gm-tab ${activeTab === 'new' ? 'active' : ''}`}
  onClick={() => handleTabChange('new')}
>
  🆕 新功能
</button>
```

2. **添加内容渲染**
```tsx
{activeTab === 'new' && (
  <div className="gm-new-feature">
    {/* 新功能内容 */}
  </div>
)}
```

3. **添加数据加载函数**
```tsx
async function loadNewFeature() {
  const res = await fetch('/api/v1/gm/new-feature')
  const data = await res.json()
  // 处理数据
}
```

### 样式定制

修改 `GMPanel.css` 来自定义样式：
- 配色方案
- 字体大小
- 间距布局
- 动画效果

## 🐛 常见问题

### 1. 按 Ctrl+G 没反应
**检查**:
- 游戏是否获得焦点
- 是否有其他快捷键冲突
- 浏览器是否支持

### 2. 面板显示空白
**检查**:
- 网络连接是否正常
- 服务端是否启动
- API 是否可访问

### 3. 操作失败
**检查**:
- 玩家 ID 是否正确
- 物品 ID 是否存在
- 参数格式是否正确

### 4. 样式错乱
**检查**:
- CSS 文件是否加载
- 浏览器兼容性
- 缓存问题
