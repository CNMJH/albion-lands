## 📋 手动测试指南

### 请在浏览器中执行以下操作：

1. **打开浏览器控制台**
   ```
   按 F12 → Console
   ```

2. **测试移动**
   ```
   右键点击游戏画面中的地面
   观察控制台输出：
   - 🖱️ 右键点击：{ x: ..., y: ... }
   - 📍 移动目标已发送
   观察角色是否移动
   ```

3. **测试攻击**
   ```
   左键点击游戏画面
   观察控制台输出：
   - ⚔️ 左键点击 - 普通攻击
   观察是否有攻击动画
   ```

4. **测试 UI**
   ```
   按 B 键 - 背包打开/关闭
   按 C 键 - 装备打开/关闭
   按 M 键 - 拍卖行打开/关闭
   按 Enter - 聊天打开/关闭
   按 Escape - 关闭所有 UI
   ```
