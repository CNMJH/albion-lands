## 🎮 交互流程

### 装备物品流程

```
1. 按 C 键打开装备面板
   ↓
2. 点击空槽位（如"主手"）
   ↓
3. 弹出物品选择窗口
   ↓
4. 选择要装备的物品
   ↓
5. 点击"装备"按钮
   ↓
6. 槽位显示已装备物品
   ↓
7. 属性实时更新
   ↓
8. 战力评分重新计算
```

### 卸下装备流程

```
1. 点击已装备的槽位
   ↓
2. 调用 unequipItem API
   ↓
3. 装备从槽位移除
   ↓
4. 物品返回背包
   ↓
5. 属性重新计算
   ↓
6. 战力评分更新
```

## 🔧 技术实现

### 状态管理

```typescript
interface EquipmentPanelState {
  isOpen: boolean              // 面板是否打开
  characterId: string          // 角色 ID
  equipment: Equipment         // 装备数据
  stats: CharacterStats        // 属性数据
  selectedSlot: EquipmentSlot | null  // 选中的槽位
  equipmentSystem: EquipmentSystem | null  // 系统实例
}
```

### 事件监听

```typescript
// 监听装备变化
useEffect(() => {
  const handleEquipmentChange = () => {
    setEquipment(system.getEquipment())
    setStats(system.getStats())
  }

  system.on('equipmentChanged', handleEquipmentChange)
  
  // 初始加载
  handleEquipmentChange()

  return () => {
    system.off('equipmentChanged', handleEquipmentChange)
  }
}, [])
```

### API 调用

```typescript
// 装备物品
async equipItem(itemId: string, slot: EquipmentSlot) {
  const result = await equipmentSystem.equipItem(itemId, slot)
  if (result.success) {
    setSelectedSlot(null)
  } else {
    alert(result.error)
  }
}

// 卸下装备
async unequipItem(slot: EquipmentSlot) {
  const result = await equipmentSystem.unequipItem(slot)
  if (!result.success) {
    console.error('卸下装备失败:', result.error)
  }
}
```

### 快捷键处理

```typescript
useEffect(() => {
  const handleKeyDown = (e: KeyboardEvent) => {
    // 聊天框激活时禁用快捷键
    const chatActive = document.querySelector('input:focus, textarea:focus')
    if (chatActive) return

    if (e.key === 'c' || e.key === 'C') {
      setIsOpen(prev => !prev)
    }
  }

  window.addEventListener('keydown', handleKeyDown)
  return () => window.removeEventListener('keydown', handleKeyDown)
}, [])
```
