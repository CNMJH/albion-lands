## 🚀 快速开始

### 1. 获取游戏

#### 方式 A: 下载 Release 包
1. 访问 https://github.com/CNMJH/albion-lands/releases
2. 下载最新版 (v0.1.0-alpha)
3. 解压到本地

#### 方式 B: 克隆仓库
```bash
git clone https://github.com/CNMJH/albion-lands.git
cd albion-lands
```

### 2. 启动游戏

#### Windows 用户
```batch
# 双击运行
launcher.bat
```

#### Linux/Mac 用户
```bash
# 终端 1 - 启动服务端
cd server
npm install
npm run dev

# 终端 2 - 启动客户端
cd client
npm install
npm run dev
```

### 3. 访问游戏
打开浏览器访问：http://localhost:3001

## 📝 反馈指南

### 报告 Bug

#### Bug 报告模板
```markdown
**标题**: [简短描述问题]

**严重程度**: P0/P1/P2/P3
- P0: 游戏崩溃、无法登录、数据丢失
- P1: 功能异常、性能问题
- P2: UI 问题、小 bug
- P3: 优化建议、小问题

**复现步骤**:
1. ...
2. ...
3. ...

**预期行为**: ...

**实际行为**: ...

**截图/视频**: (如有)

**环境信息**:
- 操作系统：Windows 11 / macOS / Linux
- 浏览器：Chrome 120 / Firefox 121 / ...
- 网络环境：WiFi / 有线
```

#### 提交方式
- **GitHub Issues**: https://github.com/CNMJH/albion-lands/issues
- **标签**: 使用 `bug` 标签

### 功能建议

#### 建议模板
```markdown
**标题**: [功能名称] 建议

**参考游戏**: (如有) 其他游戏的类似功能
```

#### 提交方式

## 🎯 测试重点

### Alpha 测试目标
1. **核心功能验证** - 确保所有核心功能可用
2. **Bug 发现** - 找出并报告严重 bug
3. **性能测试** - 验证性能指标
4. **用户体验** - 评估操作流畅度

### 重点关注
- ⚠️ 游戏崩溃或卡死
- ⚠️ 功能无法使用
- ⚠️ 数据丢失或错误
- ⚠️ 严重性能问题
