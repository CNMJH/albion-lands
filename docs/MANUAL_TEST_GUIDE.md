## 💡 测试技巧

### 1. 使用调试覆盖层

按 **F12** 打开调试控制台，可以查看：
- 玩家位置
- Canvas 焦点状态
- 网络延迟
- FPS 性能

### 2. 查看控制台日志

按 **F12** 打开开发者工具，查看 Console 标签：
- 移动日志
- 攻击日志
- 网络日志
- 错误信息

### 3. 性能监控

按 **F11** 打开性能监控：
- FPS 帧率
- 网络延迟
- 内存使用

### 4. 快捷键提示

按 **F10** 查看快捷键列表：
- 移动：WASD
- 攻击：左键/右键
- 技能：QWER
- UI: B/C/M/Escape
