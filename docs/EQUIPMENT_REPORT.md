## 🎮 核心功能演示

### 1. 装备穿戴流程

```
玩家点击装备
   ↓
检查等级是否足够
   ↓
检查物品类型是否匹配槽位
   ↓
卸下旧装备（如果有）
   ↓
更新 Character.equipment
   ↓
重新计算属性
   ↓
广播属性变化
   ↓
更新 UI
```

### 2. 属性计算

```typescript
// 基础属性（根据等级）
const baseStats = {
  attack: 10 + level * 2,
  defense: 5 + level,
  hp: 100 + level * 20,
  attackSpeed: 1.0,
  moveSpeed: 200
}

// 装备属性（累加所有装备）
const equipmentStats = {
  attack: 50,    //来自主手武器
  defense: 30,   // 来自胸甲
  hp: 120,       // 来自腿甲
  attackSpeed: 0.2,
  moveSpeed: 15
}

// 总属性
const totalStats = {
  attack: baseStats.attack + equipmentStats.attack,
  defense: baseStats.defense + equipmentStats.defense,
  hp: baseStats.hp + equipmentStats.hp,
  attackSpeed: baseStats.attackSpeed + equipmentStats.attackSpeed,
  moveSpeed: baseStats.moveSpeed + equipmentStats.moveSpeed
}
```

## 📝 技术要点

### 1. 数据一致性

- 服务端验证所有装备操作
- 事务处理（装备 + 卸下原子操作）
- 属性缓存（避免每次计算）

### 2. 性能优化

- 属性结果缓存到数据库
- 批量更新（一次换多件装备）
- 延迟计算（属性变化后 100ms 再更新）

### 3. 安全性

- 等级限制检查
- 物品所有权验证
- 槽位类型匹配
- 防止客户端作弊
