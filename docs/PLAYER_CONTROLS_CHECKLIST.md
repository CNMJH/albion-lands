## 技术架构

### 系统组件

```
┌─────────────────────────────────────────────────┐
│           PlayerControlsSystem                  │
│  ┌─────────────┬─────────────┬──────────────┐  │
│  │  键盘监听   │  鼠标监听   │  快捷键处理  │  │
│  └─────────────┴─────────────┴──────────────┘  │
│  ┌─────────────┬─────────────┬──────────────┐  │
│  │  移动处理   │  攻击处理   │  交互处理    │  │
│  └─────────────┴─────────────┴──────────────┘  │
└─────────────────────────────────────────────────┘
                      │
        ┌─────────────┼─────────────┐
        ▼             ▼             ▼
┌──────────────┐ ┌──────────┐ ┌────────────┐
│ GameRenderer │ │ Network  │ │  UI State  │
│  (事件系统)  │ │ (同步)   │ │ (Hook)     │
└──────────────┘ └──────────┘ └────────────┘
```

### 事件流

1. **输入事件** → `PlayerControlsSystem` 监听
2. **处理逻辑** → 更新本地状态 + 发送网络消息
3. **触发事件** → `GameRenderer.emit()` 通知其他系统
4. **渲染响应** → `CombatRenderer` 等监听并更新画面

### 数据流

```
玩家输入 → PlayerControlsSystem → Network.send() → 服务端
                                    ↓
                            本地状态更新
                                    ↓
                            useGameStore.updatePlayer()
                                    ↓
                            React 组件重新渲染
```
