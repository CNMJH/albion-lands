## 🚀 快速开始

### 步骤 1：拉取最新代码

```bash
# 进入游戏目录
cd albion-lands

# 拉取最新代码
git pull origin main

# 确认版本
git log --oneline -5
```

**预期输出：**
```
b575984 docs: 添加 UI 优化样式应用报告
d467cd0 feat: 应用优化样式到所有 UI 组件
...
```

### 步骤 2：安装依赖

```bash
# 安装服务端依赖
cd server
npm install

# 安装客户端依赖
cd ../client
npm install

# 返回根目录
cd ..
```

### 步骤 3：启动游戏

**方法 1：使用启动器（推荐）**
```bash
# 双击启动器
launcher.bat
```

**方法 2：手动启动**
```bash
# 启动服务端
cd server
npm run dev

# 新开命令行，启动客户端
cd client
npm run dev
```

### 常见问题

#### Q1: 编译失败

**错误：** TypeScript 编译错误

**解决：**
```bash
# 清理缓存
cd client
rm -rf node_modules
npm install
npm run build
```

#### Q2: 端口被占用

**错误：** Port 3001/3002 is already in use

**解决：**
```bash
# Windows - 查找占用端口的进程
netstat -ano | findstr :3001
netstat -ano | findstr :3002

# 杀死进程
taskkill /PID [进程 ID] /F
```

#### Q3: 数据库错误

**错误：** Prisma migration failed

**解决：**
```bash
cd server
npx prisma migrate reset
npx prisma generate
```

#### Q4: UI 样式不显示

**错误：** 样式未应用

**解决：**
```bash
# 清理浏览器缓存
Ctrl + Shift + Delete
# 或强制刷新
Ctrl + F5
```
