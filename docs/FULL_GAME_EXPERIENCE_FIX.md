## 🔧 故障排查

### 问题 1: 看不见怪物

**检查控制台日志**:
```
👾 生成怪物 slime 在 (200, 300)
✅ 成功生成 10 只怪物
```

**解决方案**:
1. 按 F12 打开控制台
2. 查找 "生成怪物" 日志
3. 如果没有，刷新页面重试

---

### 问题 2: 背包是空的

**原因**: 测试装备脚本未执行

**解决方案**:
```powershell
cd server
node scripts/add-test-items.js
```

然后刷新游戏页面。

---

### 问题 3: 角色不旋转

**检查控制台**:
```
🚶 移动中：angle: 45°
🔄 玩家旋转：45°
```

**解决方案**:
1. 确保使用**右键点击移动**（不是 WASD）
2. 检查控制台是否有旋转日志
3. 刷新页面重试

---

### 问题 4: 没有灰尘特效

**检查**:
1. 移动时观察角色脚下
2. 按 F12 查看是否有错误

**可能原因**:
- DustEffectRenderer 未正确初始化
- 浏览器性能设置限制粒子效果
