## ⚔️ 一、装备系统

### 1.1 装备决定职业

**核心理念：** "You Are What You Wear"（你穿什么，就是什么）

| 传统 MMO | 阿尔比恩 |
|----------|----------|
| 创建时选择职业 | 无职业限制 |
| 升级解锁技能 | 装备解锁技能 |
| 转职需要任务/道具 | 更换装备即可 |
| 多角色体验多职业 | 单角色体验全职业 |

**设计优势：**
```
玩家 A：今天想玩法师 → 穿法袍拿法杖 → 法师技能
玩家 A：明天想玩战士 → 穿重甲拿大剑 → 战士技能
玩家 A：后天想玩治疗 → 穿布衣拿权杖 → 牧师技能
```

### 1.2 装备分类

**武器系统：**

| 武器类型 | 主定位 | 典型武器 | 技能特点 |
|----------|--------|----------|----------|
| 剑类 | 近战 DPS/坦克 | 长剑、巨剑 | 单体/范围物理伤害 |
| 斧类 | 近战 DPS | 战斧、双斧 | 高爆发、流血 |
| 锤类 | 坦克/控制 | 战锤、连枷 | 眩晕、破甲 |
| 法杖 | 远程魔法 DPS | 火杖、冰杖、奥术杖 | 元素伤害、控制 |
| 弓类 | 远程物理 DPS | 长弓、短弓、十字弓 | 物理伤害、风筝 |
| 长矛 | 近战/控制 | 长矛、三叉戟 | 突进、控制 |
| 手套 | 辅助/特殊 | 治疗手套、毒手套 | 治疗、Debuff |
| 盾牌 | 坦克 | 塔盾、小圆盾 | 防御、保护队友 |

**防具系统：**

| 防具类型 | 定位 | 属性倾向 | 套装效果 |
|----------|------|----------|----------|
| 布甲 | 法师/治疗 | 智力、魔法值 | 魔法强化 |
| 皮甲 | 游侠/刺客 | 敏捷、暴击 | 移动速度 |
| 锁甲 | 平衡型 | 均衡属性 | 防御/恢复 |
| 板甲 | 坦克 | 护甲、生命 | 减伤、嘲讽 |

### 1.3 装备品质

| 品质 | 颜色 | 物品等级 | 属性倍率 | 掉落概率 |
|------|------|----------|----------|----------|
| 普通 | 白色 | T1-T8 | 1.0x | 60% |
| 优秀 | 绿色 | T1-T8 | 1.2x | 25% |
| 稀有 | 蓝色 | T1-T8 | 1.4x | 10% |
| 史诗 | 紫色 | T1-T8 | 1.6x | 4% |
| 传说 | 橙色 | T1-T8 | 1.8x | 0.9% |
| 神器 | 红色 | T1-T8 | 2.0x | 0.1% |

**装备等级（Tier）：**
- T1-T8：基础等级，数字越高越强
- 每个等级有 0-3 星细分
- 高等级装备需要对应资源制作

## 🌟 二、技能与天赋系统

### 2.1 命运之书（被动技能树）

**设计理念：** 提供长期成长目标，不影响职业平衡

```
                    ┌─────────────┐
                    │   中央核心   │
                    └──────┬──────┘
          ┌────────────────┼────────────────┐
          │                │                │
    ┌─────┴─────┐    ┌─────┴─────┐    ┌─────┴─────┐
    │  战斗之路  │    │  采集之路  │    │  制作之路  │
    └─────┬─────┘    └─────┬─────┘    └─────┬─────┘
          │                │                │
    ┌─────┴─────┐    ┌─────┴─────┐    ┌─────┴─────┐
    │ PVP 专精   │    │ 资源专精   │    │ 制造专精   │
    │ PVE 专精   │    │ 精炼专精   │    │ 附魔专精   │
    └───────────┘    └───────────┘    └───────────┘
```

**三条主线：**

| 路线 | 专注 | 奖励类型 | 适合玩家 |
|------|------|----------|----------|
| 战斗之路 | PVP/PVE | 战斗属性加成 | 战斗玩家 |
| 采集之路 | 资源采集 | 采集效率/产量 | 采集玩家 |
| 制作之路 | 物品制造 | 制作折扣/品质 | 制造玩家 |

### 2.2 武器技能

**每个武器提供 3 个主动技能 + 1 个被动技能：**

```
武器：火焰法杖
├── 技能 Q：火球术（单体伤害）
├── 技能 W：烈焰之地（范围持续伤害）
├── 技能 E：火焰护盾（防御 + 反伤）
└── 被动：火焰伤害 +10%
```

**技能解锁：**
- 使用武器获得熟练度
- 熟练度提升解锁技能槽
- 所有技能通用，不受等级限制

### 2.3 技能搭配策略

**经典配装思路：**

| 流派 | 武器 | 防具 | 鞋子 | 核心思路 |
|------|------|------|------|----------|
| 爆发法师 | 奥术法杖 | 学者长袍 | 法师鞋 | 高爆发秒杀 |
| 持续输出 | 自然法杖 | 牧师长袍 | 跑鞋 | 持续治疗 + 伤害 |
| 坦克战士 | 战锤 | 骑士板甲 | 重甲鞋 | 高生存 + 控制 |
| 游击刺客 | 双匕首 | 刺客皮甲 | 刺客靴 | 高机动 + 爆发 |
| 远程射手 | 长弓 | 游侠皮甲 | 游侠靴 | 安全距离输出 |

## 💥 三、战斗系统

### 3.1 战斗基础

**战斗机制：**

| 机制 | 说明 | 策略深度 |
|------|------|----------|
| 自动攻击 | 在范围内自动攻击目标 | 低 |
| 技能释放 | 手动释放武器技能 | 高 |
| 走位躲避 | 躲避敌方范围技能 | 高 |
| 技能打断 | 特定技能可打断读条 | 中 |
| 控制链 | 眩晕/沉默/减速组合 | 高 |

### 3.2 伤害类型

| 伤害类型 | 克制关系 | 典型技能 |
|----------|----------|----------|
| 物理伤害 | 被护甲减免 | 普通攻击、物理技能 |
| 魔法伤害 | 被魔法抗性减免 | 法师技能 |
| 真实伤害 | 无法减免 | 少数特殊技能 |

### 3.3 战斗公式（简化版）

```
实际伤害 = 基础伤害 × (1 - 护甲/(护甲 + 常数)) × 暴击倍率 × 其他增伤

最终伤害 = 实际伤害 - 固定减伤
```

**关键属性：**
- 物理强度 → 物理伤害
- 魔法强度 → 魔法伤害
- 护甲 → 物理减伤
- 魔法抗性 → 魔法减伤
- 暴击率 → 暴击概率
- 暴击伤害 → 暴击倍率

### 3.4 PVP 战斗特点

**1v1 决斗：**
- 考验个人操作
- 职业克制明显
- 装备质量影响大

**小规模（2v2-5v5）：**
- 团队配合重要
- 职业搭配关键
- 集火目标选择

**大规模（10v10+）：**
- 指挥协调核心
- 职业组成完整
- 战术执行关键

### 3.5 PVE 战斗特点

** solo 打怪：**
- 续航能力重要
- 选择合适目标
- 效率优先

**组队副本：**
- 铁三角配合（坦/奶/输出）
- 机制应对
- 容错率管理

## 📝 八、总结

### 8.1 核心玩法要点

1. **装备系统** - 提供职业自由度
2. **技能系统** - 简单但有深度
3. **战斗系统** - 易上手难精通
4. **探索系统** - 奖励驱动探索
