# v0.3.2-alpha 快速测试指南

**版本**: v0.3.2-alpha  
**测试重点**: UI 交互优化  
**测试时间**: 5 分钟

## 🚀 启动游戏

```bash
cd /home/tenbox/albion-lands
# 使用启动器
./launcher.bat  # Windows
# 或手动启动
cd server && npm run dev  # 终端 1
cd client && npm run dev  # 终端 2
```

**访问**: http://localhost:3001

---

## ✅ 测试清单

### 1. UI 按钮点击测试（2 分钟）

**步骤**:
1. 进入游戏，创建/选择角色
2. 依次点击以下按钮：
   - ☰ 菜单按钮
   - 👥 好友按钮
   - 📜 任务按钮
   - 🏆 成就按钮

**预期结果**:
- ✅ 所有按钮都能正常点击
- ✅ 点击后对应面板打开
- ✅ 没有"点不动"的情况

**失败情况**:
- ❌ 按钮无反应
- ❌ 点击后没反应

---

### 2. 物品 Tooltip 测试（1 分钟）

**步骤**:
1. 按 `B` 键打开背包
2. 鼠标悬停在任意物品上
3. 观察显示的 tooltip

**预期结果**:
- ✅ 显示物品名称（带品质颜色）
- ✅ 显示物品等级
- ✅ 显示物品类型
- ✅ 显示属性列表
- ✅ 显示物品描述
- ✅ 鼠标移开后 tooltip 消失

**失败情况**:
- ❌ tooltip 不显示
- ❌ tooltip 位置不对
- ❌ tooltip 不消失

---

### 3. 技能 Tooltip 测试（1 分钟）

**步骤**:
1. 观察底部技能栏
2. 鼠标悬停在每个技能图标上
3. 观察显示的 tooltip

**预期结果**:
- ✅ 显示技能名称
- ✅ 显示快捷键（如 1, 2, 3...）
- ✅ 显示技能效果描述
- ✅ tooltip 显示在技能上方
- ✅ 鼠标移开后 tooltip 消失

**失败情况**:
- ❌ tooltip 不显示
- ❌ 描述为空

---

### 4. 装备 Tooltip 测试（1 分钟）

**步骤**:
1. 按 `C` 键打开装备面板
2. 鼠标悬停在已装备的槽位上
3. 观察显示的 tooltip

**预期结果**:
- ✅ 显示装备名称
- ✅ 显示装备等级（T1-T5）
- ✅ 显示属性列表
- ✅ 显示装备作用说明
- ✅ 鼠标移开后 tooltip 消失

**失败情况**:
- ❌ tooltip 不显示
- ❌ 显示乱码
