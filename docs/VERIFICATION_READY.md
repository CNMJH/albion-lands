## 🐛 常见问题

### Q1: 编译失败

**解决：**
```bash
cd client
rm -rf node_modules
npm install
npm run build
```

### Q2: 端口被占用

**解决：**
```bash
# 查找占用进程
netstat -ano | findstr :3001
netstat -ano | findstr :3002

# 杀死进程
taskkill /PID [进程 ID] /F
```

### Q3: UI 样式不显示

**解决：**
```bash
# 强制刷新浏览器
Ctrl + F5

# 或清理缓存
Ctrl + Shift + Delete
```

### Q4: WebSocket 连接失败

**检查：**
- [ ] 服务端是否启动（端口 3002）
- [ ] 防火墙是否阻止
- [ ] 浏览器控制台错误
