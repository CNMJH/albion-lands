## 🐛 常见问题

### Q1: 客户端无法连接
**解决：**
1. 确认服务端已启动（访问 http://localhost:3002/health）
2. 检查 WebSocket 地址是否正确（ws://localhost:3002/ws）
3. 查看浏览器控制台错误信息

### Q2: 看不到怪物
**解决：**
1. 检查服务端日志是否有"生成了 10 个初始怪物"
2. 查看浏览器控制台是否有 monsterList 消息
3. 确认玩家位置在 zone_1

### Q3: 攻击没有效果
**解决：**
1. 检查与服务端的 WebSocket 连接
2. 查看服务端日志是否有攻击处理
3. 确认怪物在攻击范围内（100 像素）

### Q4: 端口被占用
**解决：**
```bash
# 杀掉占用端口的进程
kill -9 $(lsof -ti:3002)
kill -9 $(lsof -ti:3001)
```
