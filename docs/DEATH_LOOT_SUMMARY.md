## 🎮 核心规则

### 安全区域掉落规则
| 区域 | 安全等级 | 死亡惩罚 |
|------|---------|----------|
| 新手村庄 | 9-10 | 仅耐久损失 (-10) |
| 平原旷野 | 6-8 | 仅耐久损失 (-10) |
| 迷雾森林 | 3-4 | 掉落 1 件非装备物品 |
| 巨龙山脉 | 1-2 | 掉落 1-2 件装备 |
| 深渊遗迹 | 0 | 掉落 3-5 件装备 |

### 保护机制
- ✅ 新手保护：Lv.10 以下不掉落装备
- ✅ 耐久损失：死亡时所有装备耐久 -10
- ✅ 掉落物过期：30 分钟后自动删除

## 📝 API 端点

### 服务端 API
```
POST /api/v1/combat/death
  Body: { characterId, killerId?, cause: 'PVP' | 'PVE' | 'ENV' }
  Response: { success, droppedItems, respawnLocation }

POST /api/v1/combat/loot
  Body: { characterId, droppedItemId }
  Response: { success, item }

GET /api/v1/combat/drops/:mapId
  Response: { success, drops: [...] }
```
