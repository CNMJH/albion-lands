## 🎯 系统概述
死亡掉落系统是阿尔比恩风格 MMO 的核心机制，玩家在某些区域死亡时会掉落装备，增加游戏的风险性和刺激性。

## 📋 核心规则

### 安全区域掉落规则

| 区域 | 名称 | 安全等级 | 死亡惩罚 | 耐久损失 |
|------|------|---------|----------|---------|
| 区域 1 | 新手村庄 | 9-10 | **不掉落** | -10 |
| 区域 2 | 平原旷野 | 6-8 | **不掉落** | -10 |
| 区域 3 | 迷雾森林 | 3-4 | 掉落 1 件**非装备** | -10 |
| 区域 4 | 巨龙山脉 | 1-2 | 掉落 1-2 件**装备** | -10 |
| 区域 5 | 深渊遗迹 | 0 | 掉落 3-5 件**装备** | -10 |

### 保护机制

#### 新手保护
- **Lv.10 以下**玩家不掉落装备
- 鼓励新手玩家探索和学习

#### 耐久度系统
- **耐久范围**: 0-100
- **初始耐久**: 100
- **死亡损失**: -10 (所有装备)
- **耐久归零**: 装备无法使用 (待实现)
- **修理功能**: 待实现

#### 掉落物过期
- **过期时间**: 30 分钟
- 过期后自动删除
- 防止地图堆积掉落物

## 🎮 游戏操作

### 死亡流程
```
被怪物/玩家击败
    ↓
显示死亡报告 (💀 你被击败！)
    ↓
掉落物品 (黄色光圈标记)
    ↓
扣除装备耐久 (-10)
    ↓
3 秒后复活 (回到安全区)
```

### 拾取流程
```
移动到掉落物附近
    ↓
看到黄色光圈和 [E 拾取] 提示
    ↓
按下 E 键
    ↓
播放拾取动画 (缩小 + 淡出)
    ↓
显示绿色通知 "📦 拾取：铁剑"
    ↓
物品添加到背包
```

### 快捷键
- `E` - 拾取掉落物
- `B` - 打开背包
- `C` - 打开装备面板
- `Enter` - 打开聊天框

## 🔧 API 端点

### 1. 处理玩家死亡
```http
POST /api/v1/combat/death
Content-Type: application/json

{
  "characterId": "uuid",
  "mapId": "zone_4",
  "safetyLevel": 2,
  "cause": "PVE",
  "killerId": "uuid?"  // 可选
}

Response:
{
  "success": true,
  "droppedItems": [
    {
      "droppedItemId": "uuid",
      "itemId": "uuid",
      "itemName": "铁剑",
      "slot": "MainHand",
      "x": 150,
      "y": 250
    }
  ],
  "durabilityLoss": 10,
  "respawnLocation": {
    "mapId": "zone_1",
    "x": 100,
    "y": 200
  }
}
```

## 🐛 常见问题

### Q1: 死亡后没有掉落物品？
**A**: 检查以下原因：
1. 是否在安全区 (安全等级≥6)
2. 角色等级是否<10 (新手保护)
3. 背包是否为空

### Q2: 拾取掉落物失败？
**A**: 检查以下原因：
1. 距离是否过远 (>80px)
2. 掉落物是否已过期 (30 分钟)
3. 背包是否已满 (50 格)
4. 掉落物是否已被拾取

### Q3: 耐久度没有扣除？
**A**: 检查以下原因：
1. 是否穿着装备
2. 数据库迁移是否完成
3. 服务端是否重启

### Q4: 掉落物不显示？
**A**: 检查以下原因：
1. 客户端是否集成 DeathSystem
2. Canvas 渲染是否正常
3. 浏览器控制台是否有错误
