## ⚔️ 自动攻击系统

### 工作流程

```
1. 玩家右键点击敌人
   ↓
2. 检测点击位置是否有敌人（50px 范围）
   ↓
3. 计算到敌人的距离
   ↓
4a. 如果在攻击范围内 → 直接攻击
4b. 如果超出攻击范围 → 移动到攻击范围边缘
   ↓
5. 到达攻击位置后自动攻击
   ↓
6. 持续追踪目标（如果目标移动）
```

### 攻击范围检测

```typescript
// 检测点击位置是否有敌人
const clickedEnemy = detectClickedEnemy(worldX, worldY)

if (clickedEnemy) {
  // 点击到敌人 - 自动攻击
  autoAttackTarget = clickedEnemy
  moveAndAttack(clickedEnemy.x, clickedEnemy.y)
} else {
  // 点击地面 - 移动
  autoAttackTarget = null
  sendMoveTo(worldX, worldY)
}
```

### 移动并攻击

```typescript
// 计算到目标的距离
const distance = sqrt((targetX - player.x)² + (targetY - player.y)²)

// 如果已经在攻击范围内
if (distance <= attackRange) {
  performBasicAttack(targetX, targetY)
} else {
  // 移动到攻击范围边缘
  const moveDistance = distance - attackRange
  const moveX = player.x + (dx / distance) * moveDistance
  const moveY = player.y + (dy / distance) * moveDistance
  sendMoveTo(moveX, moveY)
}
```

## 🗺️ 地面网格修复

### 问题
地面一直显示深色背景，网格不显示

### 原因
`TilingSprite` 锚点未设置，位置偏移导致看不到

### 修复
```typescript
// 之前
tilingSprite.x = -(mapWidth * tileSize) / 2
tilingSprite.y = -(mapHeight * tileSize) / 2

// 现在
tilingSprite.anchor.set(0.5)  // 设置锚点为中心
tilingSprite.x = 0            // 居中放置
tilingSprite.y = 0
```

### 效果
- ✅ 深色网格背景正常显示
- ✅ 64x64 像素网格清晰可见
- ✅ 红色中心点标记每个格子
- ✅ 对角线辅助看清方向

## 🎯 攻击范围配置

### 默认配置
```typescript
attackRange: 150,    // 150 像素（LOL 风格更远）
moveSpeed: 200,      // 200 像素/秒
attackCooldown: 1000, // 1000ms 攻速（由装备决定）
stopDistance: 5,     // 5 像素内停止
```

### 攻击范围可视化
```
玩家 (0, 0)
  |
  | 150px 攻击范围
  |
  ○ ← 敌人（在范围内，直接攻击）
  
玩家 (0, 0)
  |
  | 200px 距离
  |
  ○ ← 敌人（超出范围，先移动）
     |
     | 移动到 150px 边缘
     |
     ○ ← 攻击位置
```
